# Bouchra Khabbiza - Portfolio Website

This is a personal portfolio website for **Bouchra Khabbiza**, a professional specializing in fluid studies, BIM coordination, and energy efficiency in buildings. The website showcases her expertise, experience, skills, projects, and academic background.

## Overview

The portfolio website is designed to highlight Bouchra Khabbiza's professional journey, including her:
- Work experience
- Technical skills and tools
- Completed projects
- Academic background
- Contact information

## Features

- **Responsive Design**: The website is fully responsive and adapts to different screen sizes.
- **Interactive Navigation**: Smooth navigation with a hamburger menu for smaller screens.
- **Projects Section**: Displays completed projects with images and descriptions.
- **Contact Section**: Includes email, phone, and LinkedIn links for easy communication.
- **Hero Section**: A visually appealing introduction to Bouchra Khabbiza's expertise.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout, including custom styles in `style.css`.
- **JavaScript**: For interactive elements (e.g., navigation toggle).
- **Font Awesome**: For icons used throughout the website.
- **Responsive Design**: Media queries for mobile-friendly layouts.
